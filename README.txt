Readme.txt for Virtual File System

The project is build using one header file and 4 .c files.
The MyHeader.h file consists declaration of all macros, stuctures, global variavles and functions.
The EntryPoint.c file contains the main() which is the entrypoint function of the project.
The HelperFunctions.c file contains helper functions such as opreration on string, traversal of linkedlist etc.
The FileHelper.c file consists multiple functions which will be used by Virtual File System to create, open, read write etc.
The CommandHelper.c file contains multiple functions which will be execute after commands such as ls, rm, help, man, lseek etc.
